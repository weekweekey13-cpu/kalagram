# Мессенджер

Личный мессенджер: регистрация по нику и паролю, чаты в реальном времени, контакты, поиск людей, аватарки, группы, голосовые.  
Сделан как **PWA** — удобно открывать с iPhone через «На экран „Домой“».

## Публичная ссылка (нормальный HTTPS)

Пока ПК включён, можно раздать ссылку из **`PUBLIC-URL.txt`** или запустить:

```powershell
powershell -ExecutionPolicy Bypass -File start-public.ps1
```

Откроется туннель с **настоящим HTTPS** (без «подключение не защищено»).

## Постоянный хостинг 24/7 (бесплатно)

См. **[DEPLOY.md](DEPLOY.md)** — Render / Fly.io / Railway / Docker.  
На хостинге сертификат выдаёт платформа, предупреждений нет.

## Локальный запуск (домашняя сеть)

1. Установите зависимости (один раз):

```bash
python -m pip install -r requirements.txt
```

2. Запустите сервер:

```bash
python server.py
```

3. Откройте:

- **https://localhost:8000** (ПК)
- **https://&lt;IP-ПК&gt;:8000** (телефон в той же Wi‑Fi)

Узнать IP: `ipconfig` → IPv4.

Локально сертификат «свой» — Safari может показать предупреждение → «Дополнительно» → «Перейти на сайт».  
Или откройте `http://&lt;IP&gt;:8080` и установите сертификат доверия.

## Регистрация

Любой, у кого есть ссылка, может зарегистрироваться:

- **Ник** (2–24 символа: буквы, цифры, `_` `-` `.`)
- **Пароль** (от 4 символов)

## Возможности

- Вход / регистрация
- Список чатов и непрочитанные
- Контакты + поиск людей по нику/имени
- **Групповые чаты** (кнопка 👥 в «Чаты» → название + участники)
- **Голосовые сообщения** (микрофон в чате → запись → отправить)
- Личные сообщения в реальном времени (WebSocket)
- Аватарка и отображаемое имя
- Статус «в сети» / «был(а) …»
- Индикатор набора текста
- PWA для iPhone (Safari → Поделиться → На экран «Домой»)

## iPhone: на главный экран

1. Откройте сайт в **Safari**
2. Кнопка **Поделиться**
3. **На экран «Домой»** → **Добавить**

## Данные

Хранятся локально в папке `data/`:

- `messenger.db` — пользователи, контакты, сообщения
- `avatars/` — фото профиля

## Остановка

В терминале: `Ctrl+C`
