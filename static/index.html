Микрофон на iPhone

  1. Откройте Калаграм с иконки «Домой» (не из Safari)
  2. Профиль → Разрешить микрофон → в окне нажмите Разрешить
  3. После этого Калаграм появится в: Настройки → Конфиденциальность → Микрофон
  4. Если окна не было: Настройки → Safari → Микрофон → «Спрашивать» / «Разрешить»

Пока ни разу не нажимали «Разрешить» — в списке приложений Калаграма не будет. Это нормально.

Уведомления на iPhone

  1. Откройте с домашнего экрана (не Safari)
  2. Профиль → Включить уведомления → Разрешить
