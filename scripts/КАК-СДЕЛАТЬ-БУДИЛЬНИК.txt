  Бесплатный «второй сайт»-будильник для Калаграма
═══════════════════════════════════════════════════════════

Идея: Render (Калаграм) засыпает. Ставим МАЛЕНЬКИЙ скрипт
на Cloudflare — он НЕ засыпает и раз в 5 минут будит Калаграм.

Пинг-URL Калаграма:
  https://kalagram-z20h.onrender.com/api/ping


───────────────────────────────────────────────────────────
ВАРИАНТ A — Cloudflare Worker (рекомендуем, ~3 минуты)
───────────────────────────────────────────────────────────

1. Зайди: https://dash.cloudflare.com  (регистрация free)
2. Слева: Workers & Pages → Create → Create Worker
3. Открой файл: cloudflare-ping-worker.js
   (в этой же папке scripts/)
   Скопируй ВЕСЬ код → вставь в редактор Cloudflare
4. Save and Deploy
5. Воркер → Settings → Triggers → Cron Triggers → Add
   Выражение:   */5 * * * *
   (каждые 5 минут)
6. Add / Save

Готово. Можно открыть URL воркера в браузере —
должен вернуть JSON с "ok": true.


───────────────────────────────────────────────────────────
ВАРИАНТ B — UptimeRobot (ещё проще, без «сайта»)
───────────────────────────────────────────────────────────

1. https://uptimerobot.com/signUp
2. Add New Monitor → HTTP(s)
3. URL: https://kalagram-z20h.onrender.com/api/ping
4. Every 5 minutes → Create

Это не «второй сайт», а готовый бесплатный мониторинг.
Тоже будит Render.


───────────────────────────────────────────────────────────
ВАРИАНТ C — cron-job.org
───────────────────────────────────────────────────────────

1. https://cron-job.org
2. Create cronjob
3. URL тот же /api/ping
4. Every 5 minutes


───────────────────────────────────────────────────────────
Что НЕ сработает как «не спящий» free
───────────────────────────────────────────────────────────

- Второй бесплатный Render / тот же класс хостинга — тоже уснёт
- Таймер внутри Калаграма — спит вместе с сервером
- GitHub Actions — free, но часто опаздывает на 10–20 мин


───────────────────────────────────────────────────────────
Проверка
───────────────────────────────────────────────────────────

1. Настрой A или B
2. Не заходи в Калаграм 25–30 минут
3. Открой https://kalagram-z20h.onrender.com
   → должен открыться сразу (не 40–60 сек)

Статус: https://kalagram-z20h.onrender.com/api/health
  last_external_ping_sec_ago — сколько сек назад был пинг
