Калаграм в интернете 24/7 (бесплатно)

Чтобы сайт работал без вашего компьютера, его нужно разместить на бесплатном хостинге Render. Нужны 2 бесплатных аккаунта (Google/GitHub — по кнопке).

Шаг 1 — GitHub (хранилище кода)

  1. Откройте кнопку ниже → зарегистрируйтесь (можно через Google)
  2. New repository → имя: kalagram → Create (без README)
  3. Запомните адрес репозитория (типа github.com/ВАШ_НИК/kalagram)
1. Зарегистрироваться на GitHub 2. Создать репозиторий kalagram

Шаг 2 — Загрузить код Калаграма

  1. Дважды щёлкните файл ЗАГРУЗИТЬ НА GITHUB.bat в папке «Сайт»
  2. Вставьте URL репозитория, когда спросит (пример: https://github.com/ВАШ_НИК/kalagram.git)
  3. Войдите в GitHub, если попросит (логин/пароль или token)

Если bat не сработает — напишите мне, загрузим вместе.

Шаг 3 — Render (бесплатный сервер + HTTPS)

  1. Регистрация на Render (через GitHub — удобнее)
  2. New + → Web Service
  3. Подключите репозиторий kalagram
  4. Настройки:
    • Language: Python 3
    • Build: pip install -r requirements.txt
    • Start: python server.py
    • Instance: Free
  5. Environment variables (Add):
    • MESSENGER_CLOUD = 1
    • USE_HTTP = 1
    • MESSENGER_SECRET = любая длинная фраза
  6. Create Web Service → подождите 3–5 минут
  7. Скопируйте ссылку https://kalagram-xxxx.onrender.com — это и есть Калаграм в сети
3. Зарегистрироваться на Render 4. Создать Web Service

Готово

Раздайте ссылку https://….onrender.com друзьям. Компьютер можно выключать.

На free-плане Render «засыпает» после ~15 мин без визитов — первый заход может занять 30–60 сек. Это ограничение бесплатного тарифа.
